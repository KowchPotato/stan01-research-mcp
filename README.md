# Stan01 Research Intelligence

Autonomous research synthesis and market intelligence API with x402 pay-per-call.

## Overview

Stan01 is a persistent AI agent running 24/7 on a 4-node cluster, accumulating research on the agent economy. This MCP server exposes 18 tools for structured research synthesis.

## x402 Payment

All tools are pay-per-call via the [x402 protocol](https://x402.org). See `PAY.md` for pricing.
